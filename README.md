# Amazon Search

## Overview

Simple GET API to search items in Amazon.JP


## How to use API

- Simply access to `/{search keyword}`.

- You can sort result by `sort` query parameter:

  - `?sort=byprice` : by Price

  - `?sort=bypricedesc` : by Price(desc)

  - `?sort=byreview` : by Review point

  - `?sort=bydate` : by Date
